# ToWebNovel
テキストファイルにWeb小説風の改行を加える

# これはなんや
みんな小説書いとるか？

俺は書いてへん。

せっかく書くならwebで公開したいですよね。
でも書籍の小説のような書き方をするとwebでは見づらい。

これを使うと文頭の字下げを削除して、改行ごとに空行を挿入してくれる。（字下げの削除を行うかは選択式）

対応ファイルはtxtのみ。

PySimpleGUIを使ってGUIもつけたやで。
